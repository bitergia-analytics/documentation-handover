## Basic Usage

In these videos, you'll learn how to use the interface of Bitergia Analytics Platform and
find your way among its options.

### 1: User Interface Breakdown by Sections

By following this explanation, you'll distinguish blocks and sections and make sense of
their different roles in the interaction.

[![User Interface Breakdown by Sections](https://img.youtube.com/vi/2u1ky0DJJRI/0.jpg)](https://www.youtube.com/watch?v=2u1ky0DJJRI)

### 2: Filtering Data In And Out

By following this how-to guide, you will be able to compare groups of data, zoom in on
details of a visualization, filter out excessive, unwanted, or irrelevant data, drill down
to investigate the reasons for anything that may have caught your eye, use visualization
to filter the data we want to see on another one.

[![Filtering Data In And Out](https://img.youtube.com/vi/V1yYOm6SxGU/0.jpg)](https://www.youtube.com/watch?v=V1yYOm6SxGU)

### 3: Time Picker: Selecting And Navigating Time Ranges

By following this example, you will be able to expand and narrow the time range of
visualizations to get better insight into the subject of the analysis. You likely already
know how to narrow the time range directly on the visualisation. While this is handy and
useful, it results often in random and inaccurate data visualizations that look a bit
sloppy. The time-picker allows you to be precise about the analysis' time reference points

[![Time Picker: Selecting And Navigating Time Ranges](https://img.youtube.com/vi/9QkgRdA4O3E/0.jpg)](https://www.youtube.com/watch?v=9QkgRdA4O3E)

### 4: Sharing Dashboards

In this video, you'll learn how to share a dashboard with other people using the options
offered by the interface of Bitergia Analytics platform. Following this example, you can
save a dashboard's custom status and send it as a link to others.

[![Sharing Dashboards](https://img.youtube.com/vi/do8oTdjEM0Y/0.jpg)](https://www.youtube.com/watch?v=do8oTdjEM0Y)

### 5: Data Export for External Processing and Detail Discovery

In this video, you'll learn how to extract the data used in the visualizations. Following
this tutorial, you can check data that resulted in hidden or difficult reading in graphic
visualizations. You can also extract all of the data behind the visualization and process
it elsewhere with other tools.

[![Data Export for External Processing and Detail Discovery](https://img.youtube.com/vi/FMsurZNuFq0/0.jpg)](https://www.youtube.com/watch?v=FMsurZNuFq0)
